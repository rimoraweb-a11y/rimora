# RIMORA - Agence Digitale à Nouakchott

Site web professionnel pour RIMORA, une agence digitale basée à Nouakchott, Mauritanie.

## Caractéristiques

- **Design moderne et responsive** : Optimisé pour tous les appareils (mobile, tablette, desktop)
- **Performance optimale** : Construit avec Vite et React pour une vitesse maximale
- **Tailwind CSS** : Styling moderne et efficace
- **TypeScript** : Code type-safe et maintenable
- **SEO-friendly** : Optimisé pour les moteurs de recherche

## Technologies utilisées

- **React 18** : Bibliothèque UI
- **Vite** : Build tool ultra-rapide
- **TypeScript** : Langage de programmation
- **Tailwind CSS** : Framework CSS utilitaire
- **PostCSS** : Traitement CSS avancé

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Le site sera disponible sur `http://localhost:5173`

## Build pour production

```bash
npm run build
```

## Déploiement

Ce projet est déployé sur **Vercel** et connecté à ce dépôt GitHub. Chaque push sur la branche `main` déclenche un redéploiement automatique.

## Structure du projet

```
rimora/

```
